# 🚗 AutoGear - Car Accessories Shopping Website

AutoGear is a simple car accessories e-commerce website built using **HTML, CSS, and JavaScript**.  
It allows users to browse car accessories, add them to a cart, register/login, and manage their cart.

This project is designed as a **beginner-friendly frontend project** to understand how basic e-commerce websites work.

---

## 🌐 Live Website
You can access the live website here: https://tiny-semifreddo-e89fd5.netlify.app/


---

## ✨ Features

- 🚗 Browse different car accessories
- 🛒 Add products to cart
- 🗑 Remove items from cart
- 🔐 User registration system
- 🔑 User login system
- 💾 Cart and user data stored using LocalStorage
- 📱 Responsive product grid layout
- 🎨 Modern UI with gradient colors

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage (Browser Storage)
